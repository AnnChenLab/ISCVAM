ISCVAM: Interactive Single Cell Visual Analytics for Multiomics

ISCVAM: Interactive Single Cell Visual Analytics for Multiomics, is a JavaScript-based tool for scRNA-seq data analysis. Originally developed at https://iscvam.moffitt.org/, ISCVAM has since been enhanced with extended capabilities, including the 190 TISCH datasets, and is now hosted at https://chenlab.utah.edu/iscvam/.

The computational tool ISCVAM is described in this paper:

Smalley J, Chen Z, Phadke M, Li J, Yu X, Wyatt C, Evernden B, Messina JL, Sarnaik AA, Sondak VK, Zhang C, Law V, Tran N, Etame A, Macaulay RJB, Eroglu Z, Forsyth PA, Rodriguez PC, Chen YA, Smalley KSM. Single-Cell Characterization of the Immune Microenvironment of Melanoma Brain and Leptomeningeal Metastases. Clin Cancer Res. 2021 Jul 15;27(14):4109-4125.

Briefly, it consists of two major components:

  • The First component includes a collection of Bash and R scripts (utilizing many of the widely used algorithms in the single-cell community, including:
    • Seurat for general processing,
    • SingleR for cell-type recognition, and
    • Single-cell signature explorer for gene set signature scoring that processed the scRNA-seq data offline
  • The second is the Web-based component based upon state-of-the-art technologies, including react.js from Facebook, tensorflow.js from Google and Plotly.js, that allows convenient real-time interactive exploration and hpc node.js backend was also created to serve the on-demand queries of the Web application, allowing for real-time interactive investigation of genes expressed in selected samples or subsets of cells. (A note on web-browser: ISCVA is currently available on all web browsers, except for Safari)
